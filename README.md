# FuzzyQSM
Fitting tree QSMs by quantifying and using the uncertainty inherent in the data.
The code is written to be used with MATLAB R2024b. Additional tools are included with permission:

Colorbrewer (https://github.com/DrosteEffect/BrewerMap, https://se.mathworks.com/matlabcentral/fileexchange/45208-colorbrewer-attractive-and-distinctive-colormaps)
export_fig (https://github.com/altmany/export_fig, https://se.mathworks.com/matlabcentral/fileexchange/23629-export_fig?s_tid=FX_rc2_behav)
Furthermore, it requires TreeQSM (https://github.com/InverseTampere/TreeQSM).
